# Bootstrap

Bootstrap is responsible for initializing the platform SDK when the main SDK creates its instance. Its purpose is to set up and configure the necessary components and dependencies required for the platform SDK to function correctly within the application. This ensures that any platform-specific SDKs are properly initialized and ready for use, helping to avoid potential conflicts and ensuring smooth integration.
